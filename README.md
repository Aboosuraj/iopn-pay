# 💸 IOPn Pay

A mobile-first Web3 payment application built on the **IOPn testnet**.

Users can:
- Connect crypto wallet
- Send tokens (IOPn / ERC-20)
- Receive tokens with QR code
- View transaction history

---

## 🚀 Features

- 🔗 Wallet connection (MetaMask / WalletConnect)
- 💸 Send and receive crypto instantly
- 📱 Mobile-first UI
- 📊 Transaction history (on-chain events)
- ⚡ Built for IOPn testnet

---

## 🧱 Tech Stack

- Next.js (App Router)
- React + TypeScript
- Wagmi + Viem
- RainbowKit
- TailwindCSS
- QR Code generator

---

## 🔗 Network

IOPn Testnet (EVM Compatible)
